Kuesioner Nomophobia (8 Item)

Ini self-assessment singkat tentang rasa cemas/khawatir saat jauh dari ponsel. Jawab jujur, 1 = Sangat Tidak Setuju sampai 4 = Sangat Setuju.
Bukan untuk diagnosis . Hasilnya untuk refleksi pribadi.

1) Saya akan cemas jika kehabisan pulsa atau mencapai batas data bulanan.
2) Jika tidak ada sinyal data/Wi-Fi, saya akan terus-menerus memeriksa apakah sudah ada jaringan.
3) Jika tidak membawa ponsel, saya akan merasa cemas karena tidak dapat segera berkomunikasi dengan keluarga/teman.
4) Jika tidak membawa ponsel, saya akan cemas karena tidak bisa dihubungi keluarga/teman.
5) Jika tidak membawa ponsel, saya akan gelisah karena tidak tahu bila ada orang yang mencoba menghubungi saya.
6) Jika tidak membawa ponsel, saya merasa gelisah karena akan tertinggal perkembangan terkini di media sosial/jaringan daring.
7) Jika tidak membawa ponsel, saya akan canggung karena tidak bisa mengecek notifikasi terbaru.
8) Jika tidak membawa ponsel, saya akan merasa gelisah karena tidak dapat menerima pesan teks dan panggilan.
Catatan: 8 item ini merangkum empat aspek NMP-Q.
Bagaimana cara membaca skor?
Skor dihitung dari rata-rata 8 item (rentang 1–4) dan ditampilkan juga sebagai Percent of Maximum Possible (0–100).
  • ≤ 1,75 = Rendah Low
  • 1,76–2,50 = Sedang Moderate
  • 2,51–3,25 = Tinggi High
  • > 3,25 = Sangat Tinggi Very High
Ambang di atas bersifat heuristik untuk edukasi publik (studi adaptasi tidak menetapkan cut-off klinis).
