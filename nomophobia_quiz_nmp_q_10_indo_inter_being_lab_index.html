Kuis Nomophobia (NMP‑Q‑10 • Versi Indonesia)

Nomophobia = rasa cemas saat jauh dari ponsel atau tidak bisa menggunakannya. Isi 8 pernyataan berikut (±1 menit), lalu lihat hasilmu & saran praktis.

0/8 terjawab

1. Jika tidak ada sinyal data atau Wi‑Fi, saya akan terus‑menerus mengecek apakah sudah ada sinyal/jaringan.

2. Saya akan cemas jika kehabisan pulsa atau mencapai batas kuota data bulanan.

3. Jika tidak membawa ponsel, saya akan merasa cemas karena tidak dapat menghubungi keluarga dan/atau teman‑teman.

4. Jika tidak membawa ponsel, saya akan merasa khawatir karena keluarga dan/atau teman tidak dapat menghubungi saya.

5. Jika tidak membawa ponsel, saya akan merasa gelisah karena tidak dapat menerima pesan teks dan panggilan.

6. Jika tidak dapat menggunakan ponsel, saya takut akan tersesat di suatu tempat.

7. Jika tidak membawa ponsel, saya merasa gelisah karena akan tertinggal perkembangan terkini melalui media sosial dan jaringan daring.

8. Jika tidak membawa ponsel, saya akan merasa canggung karena tidak dapat memeriksa pemberitahuan terkini.

Tentang skala ini

Ini adalah kuis skrining, bukan diagnosis. Jawab sejujur mungkin untuk melihat kecenderungan pribadi terhadap nomophobia (kecemasan tanpa ponsel).

  • Skor tiap item: 1–4 (ST‑S‑SS).
  • Total skor: 8–32. Lebih tinggi ⇒ kecenderungan lebih kuat.
  • Privasi: jawaban tidak dikirim ke mana pun dalam versi mandiri ini.
Sitasi:
